Rich Text in RCS refers to messages that go beyond simple plain text by allowing formatting and media enhancements such as: Buttons with actions.

#### Request Body Parameters

<ParamField path="number" type="string" required>
  Recipient's phone number (must be RCS-enabled and in international format) For
  example, if the number is in the India, the format would be: +91XXXXXXXXXX
</ParamField>

<ParamField path="templateName" type="string" required>
  Name of the message template to be used
</ParamField>

<ParamField path="botId" type="string" required>
  Identifier of the RCS bot used to send the message
</ParamField>

<ParamField path="type" type="string" required>
  Type of message. Typically "PRETRANS" or another predefined type
</ParamField>

<ParamField path="campaignName" type="string">
  The RCS campaign name refers to the specific name or title given to a campaign
  conducted through RCS
</ParamField>
